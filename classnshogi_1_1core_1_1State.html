Detailed Description

Manages game states in Shogi.

This class is responsible for controlling and maintaining the game state in Shogi. It includes functionality for tracking the side to move, the current piece locations (refer to the Position class for more details), applying a legal move, maintaining the game history, and handling the declaration rule.
