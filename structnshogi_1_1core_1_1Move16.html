Detailed Description

Represents any legal move in Shogi with 16 bits.

This structure encodes essential information for a Shogi move, such as source and destination, whether it is a dropping move, and promotion, in a compact 16-bit format. However, piece type information is omitted to reduce the number of used bits. This information must be retrieved from the game state when needed by nshogi::core::State::getMove32FromMove16().
